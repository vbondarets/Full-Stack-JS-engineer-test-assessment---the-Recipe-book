# Full-Stack-JS-engineer-test-assessment---the-Recipe-book

The application includes a Backend (BE) built with Node.js (Nest or Express) and a Frontend (FE) built with React

## Setup and Run backend

```bash
$ cd ./backend

$ yarn install

# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Setup and Run frontend

```bash
$ cd ./frontend

$ yarn install

# development
$ yarn run vite --host


# production mode
$ yarn run build
$ yarn run preview
```
